# 관리자 채팅 사이트

관리자 1명과 승인된 사용자 50~70명이 실시간으로 대화할 수 있는 웹사이트입니다.

## 포함 기능

- 회원가입 후 관리자 승인 방식
- 관리자/사용자 로그인
- 관리자 공지용 단체 채팅방
- 관리자와 사용자 간 1:1 실시간 채팅
- 기본 글자 크기 19px (요청 범위 18~20px)

## 실행 방법

```bash
npm install
npm start
```

브라우저에서 `http://localhost:3000` 접속
